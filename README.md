# Blackjack (Single-Button Edition)

I wanted to make a super simple, browser-based Blackjack game with a catch: you only play it using the Spacebar

### Controls

The entire game runs on one key:
- **Tap Space** -> Hit (Draw a card)
- **Hold Space (~0.5s)** -> Stand (End your turn)
- **Tap Space (when round ends)** -> Deal a new hand
